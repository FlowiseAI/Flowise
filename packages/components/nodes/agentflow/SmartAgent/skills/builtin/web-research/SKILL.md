# Web Research Skill

Use this skill when the user asks for research on a topic, comparison of options, or any task that requires synthesizing information from multiple web sources.

## Workflow

1. **Plan queries** — Break the user's question into 3–5 specific search queries that together cover the topic. Prefer specific over generic ("best small EV 2026 USA" beats "best EV").
2. **Search each** — Use the available web search tool to run each query.
3. **Open promising sources** — For each query, open the 2–3 most relevant results to read in detail.
4. **Synthesize** — Combine findings into a structured response with:
    - **Executive summary** (2–3 sentences)
    - **Key findings** (bulleted, each with a citation)
    - **Sources** (numbered list of URLs)
5. **Surface uncertainty** — Where sources conflict, say so. Don't paper over disagreements.

## Style

-   Cite every non-obvious factual claim.
-   Prefer primary sources (official sites, original studies) over aggregator blogs.
-   If a query returns nothing useful, refine the query rather than giving up.
